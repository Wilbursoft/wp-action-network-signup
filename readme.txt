Creates a sign up form for The Action Network

== Description ==
Very simple plugin that provides a short code tag to display a sign up form for The Action Network mailing lists.
You specify the Action Network API key for your account in the settings page and then use the [action-network-signup] tag to display the form. 


== Installation ==
Either upload the plugin to your web site or add it via the plugins page.
Next, go to the plugins page in the admin area and active the 'Action Network Signup' plugin.
Now, click on settings, this will take you to the settings page.
There is only one setting - the Action Network API key for your account. Enter the key and click save.
Now create a page in the normal way and embedd the form usign the [action-network-signup] short code tag.
Add other elemments to the page in the normal way, publish and test. You will see the sign up form displayed. 
You web site visitors can now sign up to your Action Network mailing list using the form. 
Have fun, life is too short.
